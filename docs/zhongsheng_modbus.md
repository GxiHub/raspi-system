## 共通設定暫存器（保持寄存器，FC06 寫入，掉電保存，重新上電生效）

| 暫存器地址 | 功能 | 備註 |
|-----------|------|------|
| **0x0032** | RS485 地址（1~255） | **地址修改用此暫存器** |
| 0x0033 | 波特率設定 | 0=4800, 1=9600, 2=14400, 3=19200, **4=38400（預設）**, 5=56000, 6=57600, 7=115200 |
| 0x003D | 奇偶校驗 | 0=無（預設）, 1=奇, 2=偶 |

## ⚠️ 新模組上線流程（必讀）

新模組出廠預設：**地址 1、波特率 38400**。  
pi52 系統使用 **9600 baud**，所以新模組接上後**無法直接通訊**，需先用 38400 連線改波特率，再改地址。

**步驟：**
1. 以 **38400 baud** 對 Slave 0x01 寫 `reg 0x0033 = 1`（改為 9600）→ 斷電重啟
2. 以 **9600 baud** 對 Slave 0x01 寫 `reg 0x0032 = 目標地址` → 斷電重啟
3. 掃描確認新地址有回應

> **注意**：若 bus 上已有其他 Slave 0x01 裝置，步驟 1 會同時影響它。  
> 建議新模組單獨接線設定完再併入 bus。

```python
import serial, struct

def crc16(data):
    crc = 0xFFFF
    for b in data:
        crc ^= b
        for _ in range(8):
            crc = (crc >> 1) ^ 0xA001 if crc & 1 else crc >> 1
    return crc

def fc06(ser, baud, slave, reg, val):
    ser.baudrate = baud
    pkt = struct.pack('>BBHH', slave, 0x06, reg, val)
    pkt += struct.pack('<H', crc16(pkt))
    ser.reset_input_buffer()
    ser.write(pkt)
    import time; time.sleep(0.4)
    resp = ser.read(8)
    return resp and resp[0] == slave and resp[1] == 0x06

ser = serial.Serial('/dev/ttyRS485', 38400, timeout=1)
# Step 1: 改波特率為 9600
fc06(ser, 38400, 0x01, 0x0033, 1)
# → 斷電重啟模組
# Step 2: 改地址（以繼電器模組二 0x0C 為例）
fc06(ser, 9600, 0x01, 0x0032, 0x0C)
# → 斷電重啟模組
ser.close()
```

## 一、數字量輸入輸出系列（繼電器/晶體管模組）

**手冊版本：** V2.1  
**用途：** pi52 系統的 Slave 0x0B（繼電器模組一）、Slave 0x0C（繼電器模組二）

### 線圈暫存器（FC01 讀 / FC05 寫單個 / FC0F 寫多個）

| 地址 | 功能 |
|------|------|
| 0x0000 | 通道 1（繼電器 1）|
| 0x0001 | 通道 2 |
| ... | ... |
| 0x002F | 通道 48 |

- 寫 1 = 開啟，寫 0 = 關閉
- FC05：`FF 00` = ON，`00 00` = OFF

### 保持暫存器（FC03 讀 / FC06 寫）

| 地址 | 功能 | 預設 |
|------|------|------|
| 0x0000~0x002F | 通道 1~48 控制（支援延時/循環等模式） | 0 |
| 0x0030 | 通訊斷線檢測時間（N×0.1 秒，0=不檢測） | 0 |
| 0x0031 | 輸入口狀態主動上傳控制 | 0 |
| **0x0032** | **RS485 地址** | **1** |
| 0x0033 | 波特率 | 4（38400） |
| 0x0034 | 批量控制（0=全關，1=全開） | 0 |

### 通道工作模式（寫保持暫存器 0x0096+）

| 值 | 模式 |
|----|------|
| 0 | 普通模式（預設）|
| 1 | 聯動模式（輸入觸發控制輸出）|
| 2 | 點動模式（翻轉）|
| 3 | 開關循環模式 |
| 4 | 自動復位模式 |

## 二、PT100 溫度採集系列

**手冊版本：** V2.0  
**用途：** pi52 系統的 Slave 0x08（PT100 溫度感測器 CH1）

### 輸入暫存器（FC04 讀，唯讀）

| 地址 | 功能 |
|------|------|
| 0x0000 | 第 1 路溫度值（CH1）|
| 0x0001 | 第 2 路溫度值 |
| ... | ... |
| 0x000F | 第 16 路溫度值 |

**資料格式：**
- 16-bit，最高位 = 符號（0=正，1=負）
- 其餘 15 位有效值，小數點 1 位（÷10 = °C）
- **0xFFFF = 感測器未接入或線路故障**

```python
raw = 0x03E8  # 讀到的原始值
sign = -1 if (raw & 0x8000) else 1
temp = sign * (raw & 0x7FFF) * 0.1  # → 100.0°C
```

### 保持暫存器（FC03 讀 / FC06 寫）

| 地址 | 功能 | 預設 |
|------|------|------|
| 0x0031 | 主動上傳間隔（×0.01s，0=不上傳） | 0 |
| **0x0032** | **RS485 地址** | **1** |
| 0x0033 | 波特率 | 4（38400） |
| 0x003D | 奇偶校驗 | 0 |
